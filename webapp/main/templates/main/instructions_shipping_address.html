Instructions setting up your Shipping Address

By default your Amazon.com Wish List Shipping Address is probably hidden. You need to select an address so {{ PROJECT_TITLE }} can send your finished item to you.

1. Go to www.amazon.com

2. Click the "List Actions" tab

3. Click the "Edit list profile" option

4. Under "Ship-to address" select an address

This is where you select which address purchases off your Wish List should be sent to.
After selecting an address, scroll down and press the "Save" button

Go to Amazon.com™.

Go back to the Home Page.
